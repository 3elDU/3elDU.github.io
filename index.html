Genetic algorithm that simulates living cells: each cell has a set of instructions (genome), essentially acting as a tiny VM. Cells can move, make energy through photosynthesis, mutate, kill each other and reproduce, just like their real counterparts.

spdl
Python CLI

Python script to download liked tracks from your spotify account. Uses youtube-dl to search and download tracks from youtube. Automatically downloads metadata for each track. When ran for 2nd, 3rd, nth time, scans for new tracks added to the library, and downloads them.
